# Personal Details Form with Live Validation

A responsive Personal Details Form built using **HTML, CSS, and JavaScript**. The project performs real-time validation, displays live error messages, validates file uploads, and prevents invalid data from being submitted.

## Features

- ✅ Live validation while typing
- ✅ Universal validation function
- ✅ Dynamic error messages
- ✅ Name validation (3–30 characters)
- ✅ Email validation
- ✅ Indian mobile number validation
- ✅ Age validation (18–100)
- ✅ Occupation validation
- ✅ Address validation
- ✅ ID proof upload validation
- ✅ File type validation (PDF, JPG, PNG)
- ✅ Maximum file size validation (2 MB)
- ✅ Responsive UI
- ✅ Clean and reusable JavaScript code

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Validation Rules

| Field | Validation |
|--------|------------|
| Name | 3–30 letters, no numbers or special characters |
| Email | Valid email format |
| Phone | Valid 10-digit Indian mobile number |
| Age | Between 18 and 100 |
| Occupation | Letters with single spaces |
| Address | Supports letters, numbers and common address symbols |
| ID Proof | PDF, JPG or PNG up to 2 MB |
